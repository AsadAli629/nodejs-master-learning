# How to Install and Run Node.js on Your Computer

---

# 1. Download Node.js  
Go to:
https://nodejs.org

Download the **LTS version** (recommended).

---

# 2. Install Node.js  
Run installer → Next → Next → Finish.

---

# 3. Verify Installation

Open Terminal / CMD and type:
```bash
node -v
npm -v


If both show versions, Node.js is installed correctly.
